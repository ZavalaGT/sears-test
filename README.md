## Deployment

Para correr proyecto.

1. Clonar repositorio.

```bash
  git clone https://github.com/ZavalaGT/sears-test.git
```
3. Cambiar a carpeta de proyecto

```bash
  cd sears-test
```

3. Instalar dependencias carpeta.

```bash
  npm install
```

4. Correr en modo desarrollo

```bash
  npm run dev
```

4. Abrir navegador

```bash
  http://localhost:5173/
```
